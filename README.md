# Task Manager Tool

A full-stack Task Manager application built using MERN Stack where users can create, update, delete, and manage daily tasks efficiently.

## 🚀 Live Demo
https://task-manager-tool-eosin.vercel.app

---

## 📌 Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Responsive UI
- REST API integration
- Full-stack architecture

---

## 🛠 Tech Stack

### Frontend
- React.js
- CSS / Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Rushi-tech11/Task--Manager--Tool.git
```

### Install Dependencies

```bash
npm install
```

### Run Frontend

```bash
cd frontend
npm start
```

### Run Backend

```bash
cd backend
npm run dev
```
