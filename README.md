# Task Manager Tool

A full-stack Task Manager application built using MERN Stack where users can create, update, delete, and manage daily tasks efficiently.

## 🚀 Live Demo
https://task-manager-tool-eosin.vercel.app

---

## 📌 Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Responsive UI
- REST API integration
- Full-stack architecture

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Rushi-tech11/Task--Manager--Tool.git
```

### Install Dependencies

```bash
cd backend
npm install
```

### Configure Environment

Create your backend environment file:

```bash
cd backend
copy .env.example .env
```

If you are on macOS or Linux, use:

```bash
cp .env.example .env
```

Update the values in `.env` (especially `MONGO_URI`).

### Run Backend

From the backend folder:

```bash
npm start
```

If your PowerShell blocks npm scripts, use:

```powershell
npm.cmd start
```

Or run the included Windows launcher:

```powershell
start.bat
```

### Run Frontend

```bash
cd frontend
npm install
npm run dev
```
